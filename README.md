# RMG Production Analytics Dashboard (Power BI)

This project analyzes textile and garment manufacturing production data using Power BI.

## Tools Used
- Power BI
- DAX
- Star Schema Data Modeling

## KPIs Implemented
- Knitting Output
- Knitting Efficiency
- Dyeing Reprocessing Rate
- Fabric Yield
- Fabric Inspection Pass Rate
- Cutting Achievement
- Sewing Efficiency
- DHU (Defects per Hundred Units)
- Finishing Achievement
- On-Time Shipment

## Dashboard Pages
1. Production Overview
2. Textile Manufacturing (TMS)
3. Garments Manufacturing (GMS)

## Features
- Interactive dashboards
- KPI cards
- Production trend analysis
- Quality performance monitoring
- Shipment performance tracking
